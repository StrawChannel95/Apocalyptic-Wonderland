# Introduction

Welcome to Apocalyptic Wonderland, a fully automatic Wabbajack list for Tale of Two Wastelands.

## What is a Wabbajack List?

A Wabbajack list is a pre-configured modlist for a game, which can be installed and run with a single click. Wabbajack is a tool that allows for the automated installation of modlists, and is the recommended way to install mods for Tale of Two Wastelands.

## What is Tale of Two Wastelands?

Tale of Two Wastelands is a mod for Fallout: New Vegas that merges the base game with Fallout 3. While retaining the original story of Fallout: New Vegas, the mod allows players to experience the events of Fallout 3 as if they were part of the original story. It also merges the gameplay styles of the two games, allowing players to use weapons and abilities from both games in the same playthrough. The TTW team has also fixed many bugs and glitches that were present in the base game, and added cut content from both Fallout 3 and Fallout: New Vegas to enhance the experience. In the eyes of many, this is the definitive way to experience Fallout 3 and New Vegas in 2024.

## What is Apocalyptic Wonderland?

Apocalyptic Wonderland is a Wabbajack list for Tale of Two Wastelands that aims to provide a seamless and easy-to-use experience for players who want to install extra mods on top of TTW. Straw (aka me) has been working on this list since the beginning of 2024. The list is designed to be as vanilla-friendly as possible, while still allowing for a high-quality modded experience. Keep in mind, this list contains a number of tweaks to the original mods, if you installed the mods in the exact same way as they are in the list, you will encounter bugs, problems and you will be missing a number of tweaks that are in the list. This list doesn't require any prior modding expertise but you will be expected to follow the installation instructions and not deviate from them. It is also mandatory to have some experience with operating a computer and using the Windows operating system.

## Requirements

:::danger Incompatible Versions 
The Epic Games, Game Pass and Microsoft Store versions are unsupported.
:::

- A legally purchased copy of Fallout 3 from either [Steam](https://store.steampowered.com/app/22900/Fallout_3/), [GOG](https://www.gog.com/game/fallout_3) or [Epic Games](https://www.epicgames.com/store/en-US/p/fallout-3). (Epic Games users have some extra steps and some steps deviate from all other versions, but the guide will clearly state when those steps are needed)
- A legally purchased copy of Fallout: New Vegas from either [Steam](https://store.steampowered.com/app/229100/Fallout_New_Vegas/), [GOG](https://www.gog.com/game/fallout_new_vegas) or [Epic Games](https://www.epicgames.com/store/en-US/p/fallout-new-vegas). (Epic Games users have some extra steps and some steps deviate from all other versions, but the guide will clearly state when those steps are needed)
- An archive manager. ([NanaZip](https://apps.microsoft.com/store/detail/nanazip/9N8G7TSCL18R?hl=en-us&gl=us) is recommended)
- Windows 10 or 11 (64-bit).
- A [Nexus Mods](https://www.nexusmods.com) account. (Premium is not required but recommended)
- A [ModPub](https://mod.pub) account.
- [Visual C++ Runtime Libraries](https://www.techpowerup.com/download/visual-c-redistributable-runtime-package-all-in-one/).
- [.NET Framework 4.8](https://dotnet.microsoft.com/en-us/download/dotnet-framework/thank-you/net48-web-installer).
- The latest driver versions for your GPU.
